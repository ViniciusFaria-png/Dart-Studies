# Calculadora Dart

Este é um projeto simples de calculadora feito em Dart, que permite ao usuário realizar operações matemáticas básicas como adição, subtração, multiplicação e divisão. O programa recebe como entrada a operação desejada e dois números, e em seguida exibe o resultado da operação.

## Funcionalidades

- Operações suportadas: `+`, `-`, `*`, `/`.
- Validação para evitar divisão por zero.
- Leitura da entrada do usuário diretamente do console.

## Pré-requisitos

Antes de rodar o projeto, você precisará ter o Dart SDK instalado na sua máquina. Caso não o tenha, siga as instruções para instalação [aqui](https://dart.dev/get-dart).
